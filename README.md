# 💰 Modern Expense Tracker

A modern **Expense Tracker Web Application** that helps users manage their daily spending, track budgets, and analyze expenses using visual charts.

## 🚀 Live Demo

https://yourusername.github.io/expense-tracker/

---

## ✨ Features

* 🔐 **Google Login Authentication**
* 💵 Add and manage expenses
* 📊 Category analytics chart
* 📈 Spending timeline chart
* 💰 Monthly budget tracking
* 📤 Export expenses to Excel
* 📱 Responsive design for mobile and desktop

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (ES Modules)
* Firebase Authentication
* Chart.js
* SheetJS (Excel Export)
